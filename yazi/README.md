# Yazi Configuration

Yazi file manager configuration with Catppuccin Mocha theming.

## Installation

Run the install script (works on macOS and Linux):

```bash
~/dotfiles/yazi/scripts/install.sh
```

Or set things up manually:

```bash
sudo pacman -S yazi stow   # or: brew install yazi stow
cd ~/dotfiles && stow -t ~ yazi
```

## Usage

Launch with `yazi` or press `-` inside Neovim (via yazi.nvim).
